# 更新日志

所有版本的更新说明都会记录在这个文件中。

## [1.0.2-alpha] - 2025-11-30

### 🎉 新增功能

- **趣味练习功能**：AI 词典页面新增词汇趣味练习入口
  - 📝 **记忆阶段**：展示词汇、释义、例句，支持自定义展示时间
  - ❓ **答题阶段**：渐进式题型（选择题→填空题→造句题）
  - 🤖 **AI 批改**：造句题由 AI 智能评分和反馈
  - 📊 **练习结果**：详细的成绩统计和题目回顾
  - 📚 **错题集**：自动收集错题，支持标记已掌握
  - 🔄 **复习计划**：基于艾宾浩斯遗忘曲线的智能复习提醒
  - 🔔 **通知系统**：练习完成和复习提醒通知

- **复习计划管理**：练习记录和错题集页面支持手动生成复习计划

### 🐛 问题修复

- 修复记忆阶段计时器与手动开始答题的冲突问题
- 修复练习结果页面得分计算和时间显示问题
- 修复题目回顾页面分数显示（每题满分 = 100/总题数）

### 🔧 技术改进

- 新增数据库迁移系统，支持增量更新
- 本地题目生成（选择题、填空题），无需 AI 参与
- 使用 SQL 计算练习时长（结束时间 - 开始时间）
- 时间显示转换为东八区时间
- 复习计划自动去重，同一词汇只保留一个未完成计划

## [1.0.1-alpha] - 2025-11-29

### 🎉 新增功能

- **桌面版顶部标题栏**：添加可拖动的标题栏区域，解决窗口无法拖动的问题
- **提示词管理增强**：新增「词典」标签页，支持管理以下提示词模板：
  - 自由写作 AI 点评提示词
  - AI 词典搜索提示词  
  - AI 词典生成提示词

### 🐛 问题修复

- **修复桌面版首页空白问题**：将 Vue Router 切换为 hash 模式，解决 Electron 环境下路由无法正确加载的问题
- **修复章节创作问题生成内容不完整**：增强 prompt 模板，确保生成完整的评估重点内容
- **修复环境描写问题生成内容缺失**：优化 prompt 模板结构，生成更完整的练习题目
- **修复情节转折问题生成角色信息不完整**：增强 prompt 模板，确保角色属性（身份、性格、动机、目标等）完整生成
- **修复 AI 点评和 AI 词典功能**：现在可以通过提示词管理页面自定义相关提示词

### 🔧 技术改进

- 优化 Electron 窗口配置，使用 `titleBarStyle: 'hiddenInset'` 实现 macOS 原生风格
- 改进提示词数据库结构，支持更多类型的提示词模板
- 后端路由现在优先使用数据库中的自定义提示词，回退到默认模板
